# 📊 채용 데이터 수집 및 분석 플랫폼

## 🧠 프로젝트 소개
채용 사이트의 데이터를 수집하고 정제하여 데이터베이스에 저장한 뒤,  
사용자가 조건에 맞는 채용 공고를 조회할 수 있도록 API 형태로 제공하는 프로젝트입니다.

단순한 데이터 수집이 아니라  
데이터 파이프라인 + 백엔드 서비스 구조를 함께 구현하는 것을 목표로 합니다.

---

## 🚀 주요 기능

### 1. 데이터 수집 (Crawling)
- 채용 공고 제목, 회사명, 링크 수집
- requests + BeautifulSoup 사용

### 2. 데이터 정제 (Processing)
- 공백 제거
- 중복 제거
- 데이터 표준화

### 3. 데이터 저장 (Storage)
- CSV 저장
- MySQL DB 저장

### 4. 데이터 조회 (API)
- 전체 채용 공고 조회
- 조건별 검색 (지역, 기술 스택)
