# 📊 채용 데이터 ETL 파이프라인 & 분석 플랫폼

---
## 🧠 프로젝트 개요

채용 사이트의 데이터를 수집 → 정제 → 저장 → API 제공까지 수행하는
**ETL 기반 데이터 서비스 프로젝트**입니다.
---

## 🎯 프로젝트 목표

| 구분      | 내용                    |
| ------- | --------------------- |
| ETL 구축  | 데이터 수집 → 정제 → DB 저장   |
| API 서비스 | FastAPI 기반 데이터 조회 제공  |
| 데이터 활용  | 통계 분석 및 대시보드 구현       |
| 확장성     | Airflow 자동화, Spark 확장 |

---

## 🛠 기술 스택

| 영역       | 기술                      |
| -------- | ----------------------- |
| Language | Python                  |
| Backend  | FastAPI                 |
| Crawling | requests, BeautifulSoup |
| Data 처리  | Pandas                  |
| Database | MySQL                   |
| 버전 관리    | Git, GitHub             |
| 환경       | venv                    |

### 📌 확장 예정

| 영역     | 기술        |
| ------ | --------- |
| 대시보드   | Streamlit |
| 자동화    | Airflow   |
| 대용량 처리 | Spark     |

---

## 🧱 시스템 구조

```
크롤링 → 정제 → DB 저장 → API → (대시보드) → (자동화)
```

---

## 🔄 데이터 흐름

| 단계 | 설명                 |
| -- | ------------------ |
| 1  | 채용 사이트 크롤링         |
| 2  | 데이터 정제 (공백, 중복 제거) |
| 3  | MySQL 저장           |
| 4  | API 제공             |
| 5  | 사용자 조회 및 필터링       |

---

## ⚡ 주요 기능

| 기능    | 설명                     |
| ----- | ---------------------- |
| 크롤링   | 채용 공고 수집               |
| 정제    | 데이터 구조화 및 정리           |
| DB 저장 | MySQL 적재               |
| 중복 방지 | UNIQUE + INSERT IGNORE |
| API   | FastAPI 기반 조회          |
| 필터    | 키워드 검색                 |
| 정렬    | 최신순 정렬                 |
